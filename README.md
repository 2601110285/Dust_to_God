# 🌌 Project: Overflow (프로젝트: 오버플로우)

> **"작은 먼지에서 시작해 다중 우주의 창조주가 되십시오."** > C언어 기초 문법과 2차원 배열만으로 구현한 텍스트 기반 우주 진화 생존 게임입니다.

| **개발 환경** | Windows Console, Visual Studio |

## 🎮 주요 기능 및 특징

- **무깜빡임 렌더링 (Flicker-Free)**: 커서 위치 제어를 통해 화면 갱신 시 깜빡임 없이 부드러운 화면 전환을 구현했습니다.
- **추격자 AI (블랙홀)**: 플레이어의 좌표를 실시간 계산하여 추격하고, 거리에 따라 질량을 흡수하는 인공지능을 적용했습니다.
- **데이터 시프트 알고리즘**: 2차원 배열 밀어내기(Shift) 기법으로 별도의 물리 엔진 없이 운석 하강 애니메이션을 구현했습니다.
- **감성적 연출**: 빅뱅 효과(Flash)와 쿨톤 컬러 기반의 ASCII 아트 은하를 통해 시각적 몰입감을 높였습니다.

---

## 🕹️ 조작법 및 규칙

### ⌨️ 조작 키
| 키 | 기능 |
| :--- | :--- |
| **방향키 (↑, ↓, ←, →)** | 플레이어 이동 |
| **숫자 키 (1, 2, 3)** | 메뉴 및 난이도 선택 |
| **ESC** | 게임 종료 및 프로그램 탈출 |

### 🚀 게임 규칙
1. 맵의 별(`*`, `,`, `G`, `S`)을 먹어 목표 질량(Mass)에 도달하면 다음 단계로 진화합니다.
2. **블랙홀(`@`)** 근처에 가면 질량을 빼앗기니 주의하세요.
3. 난이도가 높을수록 목표 질량이 커지고 포식자의 추격 속도가 빨라집니다.
